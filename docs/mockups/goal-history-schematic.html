Actions on expand: Revert sets current targets to those values and creates a new timeline entry with a why like "Reverted to Mar 10 state." Copy as preset saves it into your Goal presets (Cut/Maintain/Bulk). Edit why lets you annotate a change retroactively. The sparkline dot for the selected entry grows + turns blue.

Build notes: all source data already lives in settings.target_history. Each change writes { date, targets, why, source: 'manual'|'claude'|'preset' }. The sparkline is pure SVG, step-interpolated (not smoothed), reads targets.kcal from each snapshot. The delta-only chip row compares entry N to N-1 and only prints fields whose value actually moved. Revert creates a new history entry rather than erasing forward history, so the timeline never loses information.
