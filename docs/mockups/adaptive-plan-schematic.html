Diff view makes the week's changes obvious at a glance. Tweak box lets you add free-text feedback and re-run; pinning locks specific sessions so future weeks can't move them. Volume guardrail card shows the hard 10% ceiling is active — visible, not hidden.

Implementation: weekly cron (Sunday 8 AM local) OR on-demand button. Edge function builds the context bundle (training 4w + wellness 14d + nutrition 14d + weight 4w + pain active + goal history) → one Haiku call (~3k tokens, ~$0.003) → returns a structured plan for the next 7 days plus a rationale array. Client validates it against the safety rails client-side before showing Accept; if a rail is violated, Claude's proposal is auto-trimmed and flagged in the rationale. Pinned sessions pass through verbatim. Decline keeps last week's plan; Accept writes to plan_sessions. Every recompute is logged so you can replay the last 8 weeks of proposals and see which ones worked.
