Weekly recap runs automatically on Sunday (or on-demand). One Haiku call, ~400 tokens total. "Add as plan sessions" pipes the suggestion into the exercise-plan calendar (B panel of exercise-plan-schematic).

Implementation: a static EXERCISE_MUSCLE_MAP JSON file maps each known exercise (Run, Push-up, Squat, Deadlift, Plyometrics, Rowing, Yoga-flow...) to its primary (weight 1.0) and secondary (weight 0.4) muscle groups. Runs hit quads/hams/calves/glutes primarily; plyo hits quads/calves/core. When an exercise is imported we attach details.muscles. Aggregation is a straight sum over the current range. Claude annotation runs on a schedule (weekly) or on-demand from the heatmap card — same Ask-Coach flow with muscles data attached.
